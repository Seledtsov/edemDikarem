Внешний вид страниц

Страницы в системе могут иметь одну, две или три колонки. В АРМ "Страницы" есть поле: "Показывать". Для него можно выбрать один из трех вариантов значения:

  • Одну колонку
  • Две колонки
  • Три колонки
Страница всегда показыватеся в соответствии с тем, какое значение установлено в этом поле.

Страницы, что будет показано

В АРМ страницы есть следующие поля:

Название - там вы вводите название страницы для себя. Нигде, ктоме этого АРМ, вы его не увидите.

Показывать - определяет, сколько колонок будет показано при просмотре старницы. При значении "Одну колонку" будет показан текст из поля "Первая колонка", при значении "Две колонки" будет показан текст из полей "Первая колонка" и "Вторая колонка", при значении "Три колонки" будет показан текст из полей "Первая колонка", "Вторая колонка" и "Третья колонка".

Навигация - устанавливает связь страницы с разделом сайта. Страница может быть показана только в том разделе, с которым она связана. Сделано это для того, чтобы разделы знали, какую страницу показать, когда в них тыкает юзер.

Порядок - в этом поле можно задать числовое значение. Оно нужно для того, чтобы раздел мог выбрать страницу, если у него из несколько. Будет показана страница с наименьшим числом в этом поле.

Публикация - определяет, будет ни страница показана. Только страницы, у которых в этом поле стоит "Опубликовано", могут быть увидены пользователем. Страница со словами "Не опубликовано" не будет увидена пользователем ни при каком раскладе.

Jumbotron. Название - определяет название центрального блока на странице, если он есть. Если он вам не нужен, оставьте поле пустым.

Jumbotron. Текст - определяет текст центрального блока на странице, если он есть. Если он вам не нужен, оставьте поле пустым.

Первая колонка, вторая колонка, третья колонка - тут все просто, колонки старницы, считаются слева направо.

Ссылка на другую страницу

Чтобы поставить ссылку на другую страницу, вам нужно знать три вещи:

  1. Номер раздела навигации, в котороми должна быть показана страница.
  2. Номер страницы, которая должна быть показана.
  3. Как склеить эти номера в правильную ссылку.
Помните, что для успешного показа эта страница должна быть связана с этим разделом.

Номер страницы виден в АРМ "Страницы". По аналогии - номер раздела навигации виден в АРМ "Навигация сайта".

Предположим, вы хотите показать страницу "Пример статьи" (номер 4) в разделе "Главная" (номер 26).

Чтобы ссылка была работающей, вам надо получить из цифр 26 и 4 вот такую строку: ?nav_id=26&id=4. Если вам нужна ссылка на другой раздел и другую страницу, просто подставьте в этот пример нужные цифры.

Чтобы вставить ссылку в страницу нужно:

  1. Написать текст, который должен быть ссылкой.
  2. Выделить этот текст и нажать на кнопку для вставки ссылки на тулбаре редактора.
  3. На вкладке "General" появившегося диалога в поле "Link URL" указать строку ссылки. Для нашего примера это строка ?nav_id=26&id=4
  4. Нажать кнопку "Insert" диалога.

Ссылка в виде кнопки

У вас есть возможность вставить в текст красивую кнопку, типа такой:

Для того, чтобы этого добиться, нужно:

  1. Написать текст, который вы хотите видеть на кнопке.
  2. Выделить эти слова и нажать на кнопку для вставки ссылки на тулбаре редактора.
  3. Указать адрес страницы, куда ссылка должна вести, на вкладке General появившегося диалога.
  4. На вкладке Advanced заполнить поле Classes текстом "btn btn-primary" (без кавычек).
  5. Нажать кнопку "Insert" диалога.

После этого на морде будет видна красивая кнопка. Еслди вы не боитесь редактировать HTML, можете сделать все там, в помощь вам следующий фрагмент:

<a href="?nav_id=26&id=4" class="btn btn-primary">Подробнее &raquo;</a>
